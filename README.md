# Insurance Analysis

Detta projekt innehåller en analys av försäkringskostnader med hjälp av R.
Analysen inkluderar visualiseringar och regressionsmodeller för att förstå vilka faktorer som påverkar försäkringskostnader.

## Filer
- analysis.R (R-kod)
- Försäkringskostnader.pdf (rapport)
- insurance_costs.csv (data)
